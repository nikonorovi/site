How to add disqus comments on your jekyll project

Dec 6, 2017 14:08 · 187 words · 1 minute read

Add disqus in your jekyll project

  • The first thing you need is register on https://disqus.com/

  • Add URL of your project to disqus here

  • Find disqus code here Don’t forgot change your-url to your.

Example:

<div id="disqus_thread"></div>
<script>
    /**
     *  RECOMMENDED CONFIGURATION VARIABLES: EDIT AND UNCOMMENT THE SECTION BELOW TO INSERT DYNAMIC VALUES FROM YOUR PLATFORM OR CMS.
     *  LEARN WHY DEFINING THESE VARIABLES IS IMPORTANT: https://disqus.com/admin/universalcode/#configuration-variables
     */
    /*
    var disqus_config = function () {
        this.page.url = PAGE_URL;  // Replace PAGE_URL with your page's canonical URL variable
        this.page.identifier = PAGE_IDENTIFIER; // Replace PAGE_IDENTIFIER with your page's unique identifier variable
    };
    */
    (function() {  // DON'T EDIT BELOW THIS LINE
        var d = document, s = d.createElement('script');
        
        s.src = 'https://your-url.disqus.com/embed.js';
        
        s.setAttribute('data-timestamp', +new Date());
        (d.head || d.body).appendChild(s);
    })();
</script>
<noscript>Please enable JavaScript to view the <a href="https://disqus.com/?ref_noscript" rel="nofollow">comments powered by Disqus.</a></noscript>

Don’t forgot to change your-url in code

  • Edit your layout “for me is _layouts/post.html”, end add code in the bottom of the file after

  • Add in your each post varible “comments: true”

like this:

---
layout: post
`comments: true`
title:  "Some post"
author: "Is me"
---
